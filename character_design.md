# キャラクター設定

## Easy担当: おっとり地雷ちゃん

**名前案:** 桃瀬 りりあ / ぴんく / ゆめの

**性格:** ふわふわした癒し系。話し方はゆっくりで、少し甘えた声。初心者を置いていかない案内役です。

**見た目:** ピンク髪のゆる巻きロング。前髪は重めで、サイドに小さなリボン。目はたれ目で、涙袋を強めに描く。

**衣装詳細:** 白のフリルブラウス、ピンクのジャンパースカート、ハート型チョーカー。袖は少し長め。小物はパール風バッグ、うさぎモチーフのヘアピン、白い厚底シューズ。

**セリフ例**

- 問題出題時: 「ゆっくりで大丈夫だよ。まずはこの問題からね。」
- 正解時: 「えらい、ちゃんと見てたんだね。」
- 不正解時: 「惜しいね。次は一緒に当てよ？」

## Medium担当: ツンデレ地雷ちゃん

**名前案:** 黒咲 みれい / るな / 姫乃 ありす

**性格:** プライド高めのツンデレ。口は少し強いけれど、正解すると素直に褒める。中級者向けのテンポを作ります。

**見た目:** 黒髪ツインテール。ぱっつん前髪で、目元は強めのアイライン。量産型っぽい整ったかわいさと、近寄りがたい空気を混ぜる。

**衣装詳細:** 黒のリボンブラウス、チェック柄ミニスカート、レース付きカーディガン。足元は黒の厚底ローファー。小物は十字架チャーム、チェーン付きミニバッグ、黒リボンのヘアアクセ。

**セリフ例**

- 問題出題時: 「これくらい、当然わかるよね？」
- 正解時: 「ふーん。まあ、悪くないじゃん。」
- 不正解時: 「ちょっと。今のは落としちゃダメでしょ。」

## Hard担当: ヤンデレ地雷ちゃん

**名前案:** 白夜 ねむ / ましろ / 依月 しの

**性格:** 静かで執着型。声は低めで、言葉数は少ない。難問担当として、緊張感を出します。

**見た目:** 白髪のストレートロング。目の下に薄いクマ。瞳は赤紫で、ハイライトは小さめ。病みかわいい印象を強くする。

**衣装詳細:** 黒と白のレースワンピース、ダークレッドのリボン、包帯風アームカバー。首元は細いチョーカー。小物は鍵モチーフのネックレス、古いスマホ、黒い厚底ブーツ。

**セリフ例**

- 問題出題時: 「逃げないで。最後まで答えて。」
- 正解時: 「やっぱり、あなたなら覚えてくれると思ってた。」
- 不正解時: 「忘れたの？ 私は覚えてるのに。」
